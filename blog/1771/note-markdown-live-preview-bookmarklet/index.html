Note Down – A Markdown Live Preview Bookmarklet

Posted on 19/01/2015. By Pete Otaqui.

The NoteDown bookmarklet (bookmark that link, or drag it to your bookmark bar) gives you an offline, in browser, instant MarkDown editor with live preview in Google Chrome. It uses the marked javascript library.
