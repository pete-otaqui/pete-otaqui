Firebug 1.5 updates with cool new context-sensitive “Break On Next”

Posted on 23/02/2010. By Pete Otaqui.

Firebug, the mighty hammer of web development, has updated to 1.5. The coolest new feature is the “Break On Next” button is now context sensitive, depending on which panel you are looking at. The breakdown of “break on next …” to panels is as follows:

Firebug Break on Next Specifics
Panel Event
Console Break on All Errors
HTML Break on mutate
CSS Disabled
Script Break on Next statement
DOM Disabled
Net Break on XHR

It’s really great to see this extension continue to improve not only in terms of stability and useful features without getting bloated.

The original note about this feature is on the Firebug 1.6a5 blog post .
