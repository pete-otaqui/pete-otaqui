Text Colours for Readability and Firefox

Posted on 03/04/2007. By Pete Otaqui.

It seems, according to some research into readability by Alyson Hill of Austin State University, that green text on a yellow background is the most ‘readable’. The specific hex values she suggests are:

  • color:#006600;
  • background-color:#ffffcc;

I’m now using these as the default colours in a page where none are specified (in itself this is an interesting exercise in testing stylesheets). Firefox has a weird setup whereby you actually have to go into about:config in order to use any colours except a rather poor set of choices from the browser settings dialog box. The keys for those are: browser.display.foreground_color and browser.display.background_color – once there you can just the hex values above.
