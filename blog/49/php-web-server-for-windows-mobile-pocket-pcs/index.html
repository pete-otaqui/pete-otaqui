PHP Web Server for Windows Mobile Pocket PCs

Posted on 02/04/2007. By Pete Otaqui.

Pocket HPH is a project aiming to develop a windows-mobile powered php-compatible web server. Along the lines of Pocket ASP and vxWeb this will be a dynamic database-enabled (SQLite) web server on the WinCE platform. The project seems to be having trouble getting indexed by search engines with it’s name not being ‘Pocket PHP’ (note that’s a shameless attempt at remedying the issue ;)

It also has the downside of the developer offering completely free but time-limited betas – the theory apparently being to encourage people to always update, but the reality being annoying once you’ve built an application and then it starts complaining that it wants to be upgraded.
